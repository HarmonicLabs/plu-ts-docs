# OutputDatum

Represents the datum associated with a transaction output.

## `NoDatum`

Indicates there is no datum attached.


## `DatumHash`

Represents a datum referenced by its hash.

| Field | Description               |
| ----- | ------------------------- |
| `hash`  | 32-byte hash of the datum |


## `InlineDatum`

Represents a datum provided inline.
